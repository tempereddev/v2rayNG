## v2rayNG-Pro 1.0.5

This release makes updates easier to trust by showing exactly what changed, both inside the app and on GitHub.

### What changed in the app
- Added a bundled release-notes catalog so the installed build can explain itself inside the app.
- Added a one-time "What's new" dialog after version changes.
- Expanded the About screen with current-build highlights and a direct release-history link.
- Improved the update screen with published date, asset count, and release channel details.

### What changed in release management
- Added a managed changelog for version-to-version tracking.
- Updated GitHub Actions to publish releases with a proper body instead of empty release pages.

### Upstream transparency
- This fork still identifies its upstream base version and upstream commit in About.
- Release notes now make it clearer whether a change came from the fork, the updater, or the release pipeline.
